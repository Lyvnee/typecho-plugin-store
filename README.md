## Typecho-plugin-store
 

> 收集一些常用的Typecho插件

### 使用帮助

- 下载想要使用的插件，上传至站点`/usr/plugins/`目录，在后台`控制台`->`插件`中启用插件，并按说明进行相关设置即可。
- 各插件的具体使用方法，请查阅插件目录中`README`文件。
